# JellyFish Lighting - James & Company Lighting

A beautiful, responsive website for JellyFish Lighting services by James & Company Lighting in Fresno, CA.

## About

This website showcases the premium outdoor LED lighting solutions offered by James & Company Lighting, an authorized JellyFish Lighting dealer with over 40 years of experience serving the Central Valley, CA area.

### Features

- **Permanent Outdoor Lighting**: Professional installation of JellyFish Lighting systems
- **Color-Changing LEDs**: 16 million color options controlled via smartphone app
- **Smart Home Integration**: Works with Alexa, Google Home, and Control4
- **Year-Round Solutions**: Accent lighting, holiday displays, and security lighting
- **Professional Installation**: Expert local service with 40+ years of experience

### Technologies Used

- HTML5
- CSS3 (with custom animations and responsive design)
- Vanilla JavaScript
- Google Fonts (Montserrat)

### Key Features

- **Hero Slideshow**: Automatic image rotation showcasing different lighting scenarios
- **Lead Generation Form**: Contact form for free consultations
- **Responsive Design**: Mobile-friendly layout with sticky CTA
- **Interactive Elements**: Smooth animations and hover effects
- **Professional Branding**: Consistent with James & Company Lighting's established aesthetic

## Installation

This is a static website that can be served directly from any web server or hosting provider.

```bash
# Clone the repository
git clone https://github.com/Whitehurst87/jellyfish-lighting.git

# Serve locally (requires Python)
python -m http.server 8000

# Or use any local server of your choice
```

## Deployment

This site is configured for deployment on GitHub Pages. The main branch serves as the source for the live site.
